Você não pode criar uma variável em Go e não utiliza-la
Essa é uma das formas de deixar o código limpo e correto, sem desperdício de memória.

Você também não pode redeclarar uma variável dentro da mesma função

Go é case sensitive, então a variável "A" é diferente da variável "a"

Sempre coloque acrônimos em uppercase.
ex: http -> HTTP
    Url -> URL

###########################################################
Você pode fazer conversão de tipos entre variáveis.
ex:

var i float32 = 42

var j int
j = int(i)

O retorno de i será como tipo in ao invés de tipo float32.
###########################################################

Tipos de variáveis:
uint = somente inteiros positivos
int = inteiros positivos e negativos
float = numeros quebrados positivos e negativos
string = caracteres

Tipos de operadores:
-> Aritméticos
soma: +
subtração: -
divisão: /
modulo: %
multiplicação: *
incremento: ++
decremento: --
potência: <<
divisão de potência: >>

-> Relacionais
diferença: !=
comparação: ==
maior: >
menor: <
maior igual: >=
menor igual: <=

-> Lógicos
and = &&
or = ||
nor = ^
and not = &^
negação = !=

-> Atribuição
Atribuição: =
Soma e Atribuição: +=
subtração e Atribuição: -=
multiplicação e Atribuição: *=
divisão e Atribuição: /=
modulo e Atribuição: %=
